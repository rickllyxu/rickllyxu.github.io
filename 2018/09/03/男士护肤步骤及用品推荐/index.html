男士护肤步骤及用品推荐

Posted on 2018-09-03

男士护肤步骤及用品推荐：

护肤过程总结

早：

洗面奶 -> 爽肤水/纯露 -> 无油保湿乳液/或精华 -> 防晒霜

晚：

卸妆油清洁 -> 洗面奶 （一周一次搭配洁面刷） -> 爽肤水/纯露 -> (可选：眼霜) -> 无油保湿乳液/或精华

  1. （晚）卸妆油
  2. 洁面。洗面奶
  3. 补水。爽肤水
  4. (optional)去角质。水杨酸/果酸
  5. 保湿。面霜/乳液
  6. 防晒。

护肤步骤详解 reference link

1. 清洁：

第一步：卸妆 — 不要认为只有化妆才需要用卸妆油！

卸防晒。去黑头白头。例如：Fancl的纳米卸妆油

第二步：洗面奶 — 选对洗面奶

特别是毛孔粗大，黑头白头严重的朋友，最好使用泡沫丰富的洗面奶，有必要的话，可以使用起泡网。推荐：Fancl男士洗面奶 ；Fancl基础洗面奶

第三步：洁面刷 — 毛糙的肌肤怎么破？（optional）

略

第四步：爽肤水 — 化妆棉上还能擦出好多脏东西！

用化妆棉沾上爽肤水做二次清洁，完成洁面的最后一步。

这一步我通常用纯露，因为成分简单刺激小，而且纯露往往便宜大碗，随便用不心疼。

我比较推荐玫瑰纯露，即便敏感肌都可以用，而且使用感非常清爽舒服。

这可能跟油性皮肤有关吧，讨厌一切黏腻质地的东西。

顺便推荐一个化妆棉，日本的尤妮佳。

这个牌子的化妆棉分好几种，各有用途。

我比较推荐图中这款，做二次清洁非常适合，薄厚适中，吸水能力也不错。

左：Floridan爽肤水 右：尤妮佳化妆棉

2. 保湿：

我推荐的保湿产品是啫喱状的，非常之轻薄，使用体验感也非常好！韩国Dr.Jart：无油保湿啫喱

3. 防晒：

防晒很重要…… 推荐：资生堂男士防晒

QA

皮肤类型甄别：

油性？干性？混合性？中性？敏感性？参考：如何分辨油皮、混油皮、混干皮、干皮、敏感皮肤？

洗面奶类型：

分类方法很多。

  • 泡沫型和无泡沫型。泡沫型洗面奶含有表面活性剂，特点是起泡多，易冲洗。通过表面活性剂使油脂在水中稳定存在，溶于水达到清洁效果，清洁力较强。无泡沫型加入了适量的油相和少量表面活性剂，较为温和，适合干性皮肤和秋冬季的混合型皮肤。
  • 氨基酸型（弱酸性）和皂基型（弱碱性）。 氨基酸型的ph和人脸接近，较为温和，建议干性皮肤选择氨基酸型洗面奶。皂剂产品去脂力强，又容易冲洗，洗后肤感非常清爽，适合油性皮肤。

用品推荐

  1. 洗面奶：

    • lab 朗仕多功能洁面乳
    • Fancl男士洗面奶
    • Fancl基础洗面奶

  2. 爽肤水：

    • MUJI 无印良品：这个系列分舒爽型、滋润型和高保湿型。粘稠度依次是高保湿型＞滋润型＞舒爽型。推荐冬天高保湿，春秋滋润，夏天舒爽。干性皮肤推荐一年四季高保湿。
    • 理肤泉补水喷雾
    • 城野医生毛孔收敛水： 适合有痘痘的情况，贴脸

  3. 面霜/乳液：

    • innisfree悦诗风吟绿茶面霜

      质地，不黏腻，夏季完全可以使用，容易吸收，对痘痘也有镇静缓解的作用，保湿效果很好。

    • Curel珂润保湿面霜

      质地不粘腻，保湿度很高，含有角鲨烷，蓝桉叶提取，还有神经酰胺，能帮助修复皮肤屏障，非常适合秋冬干燥季节使用。Milk目前在用的就是这款Curel的水和霜，推荐。

    • 倩碧黄油无油型

      明星产品，销量极高的倩碧小黄油。几户无味，质地偏油但用起来却不觉得油，能有效保持皮肤原有的水油比例，效果持久，白天一天都觉得皮肤细腻舒服。牛奶很少用乳液，但这款小黄油，着实推荐。

    • LAB男士清爽保湿啫喱

      适用于：如果你的皮肤比较油又爱长痘痘，那建议尽量不要涂含有封闭剂的保湿产品。选择无油配方的保湿啫喱或者霜就行了。

  4. 防晒：

    • 安耐晒

  5. 护手霜：

    • 资生堂美润护手霜

      冬季天气干燥，护手霜还是要擦的。另外，睡前涂护手霜对手部皮肤增白、减少手部细纹效果很好，相信我。

  6. 卸妆油：

    • Fancl的纳米卸妆油
    • Eve Lom 的卸妆膏

对症下药

去角质

如果你的皮肤角质层不能正常代谢的话，就会导致皮肤暗沉、粗糙起皮等问题，甚至形成大量痤疮、黑白头。而使用酸类产品可以帮你快速剥脱表层老化角质层，保持肌肤的光滑、柔软。

  • 水杨酸（BHA）：比较温和的，它是一种脂溶性有机酸，所以特别适合油性和痤疮性皮肤的盆友使用，适用于清理局部性黑白头、痤疮。
    • 推荐这款宝拉珍选的2%水杨酸面部精华液，2％这个浓度的水杨酸用于面部来说比较合适。每次使用4-5滴低于化妆棉上，避开眼周肌肤涂抹即可。
  • 果酸
    • 宝拉珍选（宝拉大妈的酸的确很厉害）。这款果酸面部精华有着淡化痘印、提亮肤色的作用。另外，8％的果酸浓度并不算低，使用后一定要注意保湿！

抗衰

祛痘

毛囊正常至严重：正常毛孔–》黑头–》白头–》丘疹–》囊肿–》结节

白头和黑头时期： 痤疮幼年时期。这个阶段可以用角质溶解成分为主的外用护肤品，基本就是果酸、水杨酸、VA酸类这几样（孕妇忌用）

  • 【宝拉珍选】经典水杨酸 【价格】179/60ml
  • 果酸：修丽可果酸焕活精华/Dr WU杏仁酸焕肤精华/ 【价格】780/30ml
  • A酸也用在抗老产品中 【维A酸乳霜】 （但看起来使用有点麻烦）

即使大家手欠挤了痘痘，记得挤完迅速洗脸，然后选用一些抗炎抗敏感的护肤水。如果这些水/纯露里含有：马齿苋/金缕梅/金盏花/牛蒡/玫瑰/薰衣草等都是对消炎抗敏修复痘痘肌有功效的。

  • 契尔氏的金盏花水 【成分】金盏花，可消炎并愈合伤口；药蜀葵，可抗发炎，抗菌；牛蒡，可解除皮肤病的毒素，有收敛效果。 【价格】330/200ml
  • Dreamtimes梦幻爽肤水 【成分】马齿苋提取物润滑皮肤、抑制干燥引起的死皮和角质层的生成；生物碱和黄酮：消炎抗菌，预防发炎，可以收敛强化肌肤； 【价格】93/150ml DT梦幻水一度让我看到了国货护肤潮牌的希望，每天晚上坚持湿敷，半个月我脸颊高原红的地方，红血丝很重，居然有消退现象，不是很敏感了。绝对是列入回购名单里的一款爽肤水。
  • 法国Florihana金缕梅or玫瑰纯露 不同于爽肤水，纯露是天然植物蒸馏提取，所以必须要在一个月里用完，200ml的量建议湿敷。不管是玫瑰还是金缕梅，对痘痘都有强收敛，镇定肌肤的作用。 【价格】：130/200ml
